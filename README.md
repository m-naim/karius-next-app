# Boursehorus

This is a [Next.js](https://nextjs.org/), [Tailwind CSS](https://tailwindcss.com/) app.

## Quick Start Guide

## Installation

```bash
yarn
```

## Development

First, run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
